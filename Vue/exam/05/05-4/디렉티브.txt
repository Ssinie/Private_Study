디렉티브란.

HTML 태그 안에 v- 접두사를 가지는 모든속성을 의미함.

v-if
 - if문을 vue 디렉티브로 표현할 때 사용
v-for
 - for문을 vue 디렉티브로 표현할 때 사용
v-show
 - if와 유사하지만 조금 상이함.
 - if는 false일 경우 해당 태그를 완전 삭제함.
 - show는 css효과만 'display:none'로 주어 실제 태그는 남아있고 화면 상에서 보이지만 않게 됨.
v-bind
 - HTML 태그의 기본 속성과 뷰 데이터 속성을 연결
v-on
 - 화면요소의 이벤트를 감지하여 처리할 때 사용함.
 - v-on:click은 해당 태그의 클릭 이벤트를 감지하여 특정 메서드를 실행할 수 있음.
v-model
 - 폼(form) 에서 주로 사용함.
 - 폼에 입력한 값을 뷰 인스턴스의 데이터와 즉시 동기화 함.
 - 서버에 보내거나 watch와 같은 고급 속성을 이용하여 추가 로직을 수행할 수 있음.
 - <input>, <select>, <textarea> 태그에만 사용할 수 있음.
